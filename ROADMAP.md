# 🗺️ Product Roadmap - NeuroTime

This roadmap outlines the evolution of NeuroTime from a simple mission tracker to a complete freelance business suite.

## ✅ Phase 1: MVP (Completed)
*Primary Goal: Core mission tracking and earnings calculation.*

- [x] Basic CRUD for missions.
- [x] Automated daycare/nighttime rate calculation.
- [x] Simple statistics (total revenue, hours).
- [x] Supabase integration for data persistence.
- [x] Basic Email/Password Authentication.
- [x] Glassmorphism UI components.

## 🚀 Phase 2: V1 - Professional Tool (Current)
*Primary Goal: Enhanced utility and productivity.*

- [x] **Goal System**: Set monthly and yearly targets.
- [x] **AI Assistant**: Integrate Google Gemini for professional description enhancement.
- [x] **PWA Support**: Offline mode and homescreen installation.
- [x] **Smart Dashboard**: Visual progress bars and activity breakdown.
- [x] **Client Auto-completion**: Smart suggestions for client names.
- [x] **Responsive Design**: Optimized for mobile, tablet, and desktop.
- [x] **Multi-slot Missions**: Support for complex schedules in a single mission.

## 🛠️ Phase 3: V2 - Financial Optimization (Coming Soon)
*Primary Goal: Financial tracking and simple billing.*

- [ ] **Invoice Generation**: Generate PDF invoices directly from completed missions.
- [ ] **Expense Tracking**: Add business-related expenses (travel, equipment).
- [ ] **Tax Estimation**: Calculate estimated social contributions based on status (Auto-entrepreneur FR).
- [ ] **Data Export**: Export mission history as CSV/Excel for accounting.
- [ ] **Dark/Light Mode**: Full theme customizer.
- [ ] **Multi-currency Support**: Switch between EUR, USD, etc.

## 🌐 Phase 4: V3 - Ecosystem & Growth (Future)
*Primary Goal: Scalability and advanced features.*

- [ ] **Shared Missions**: Ability to share mission details with co-workers.
- [ ] **Advanced AI Insights**: Revenue forecasting and peak activity analysis.
- [ ] **Calendar Integration**: Sync missions with Google Calendar or iCal.
- [ ] **Contract Template**: Generate simple legal contracts for missions.
- [ ] **Mobile App**: Native iOS/Android apps via Capacitor or React Native.

---

*Note: This roadmap is subject to change based on user feedback and development priorities.*
